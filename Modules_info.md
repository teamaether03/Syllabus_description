# Module Overview

The **Technical Foundation** is divided into 11 modules. Each module builds a core engineering skill, and together they form the foundation for designing, building, securing, and scaling modern technology.

| Module                               | One-Line Description                                                                                                                  |
| :----------------------------------- | :------------------------------------------------------------------------------------------------------------------------------------ |
| **1. Programming (Python, C & Git)** | Learn how to think logically, write code, manage projects with Git, and solve problems using programming.                             |
| **2. Linux**                         | Learn how computers are operated from the terminal, manage files, processes, users, and automate tasks.                               |
| **3. Computer Architecture**         | Learn how a computer works internally—from CPU and memory to how instructions are executed.                                           |
| **4. Operating Systems**             | Learn how an OS manages hardware, processes, memory, files, and enables applications to run.                                          |
| **5. Networking**                    | Learn how devices communicate over the internet using IPs, protocols, DNS, HTTP, routing, and networking fundamentals.                |
| **6. Databases**                     | Learn how applications store, organize, retrieve, and manage data efficiently using SQL and database systems.                         |
| **7. APIs**                          | Learn how different applications communicate by sending requests, exchanging data, and integrating services.                          |
| **8. Cloud Computing**               | Learn how modern applications are deployed, hosted, and scaled using cloud infrastructure and services.                               |
| **9. AI Fundamentals**               | Learn how modern AI models work, how to use them effectively, and integrate AI into applications and workflows.                       |
| **10. Cybersecurity**                | Learn how to build secure systems by understanding threats, vulnerabilities, authentication, encryption, and security best practices. |
| **11. System Design**                | Learn how to design reliable, scalable, and maintainable software systems by connecting all the above concepts together.              |
